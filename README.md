### Vosk TTS Mobile

Мобильное приложение на анроид для запуска модели синтеза VoskTTS

#### Инструкция по сборке

Проект готов к сборке.
Можно использовать Android Studio для запуска на физическом или виртуальном устройстве в режиме отладки.
Самый простой способ получить .apk: выполнить команду graldew assembleRelease - тогда apk будет в app/build/outputs/apk/release
