### Vosk TTS Mobile

Мобильное приложение на анроид для запуска модели синтеза VoskTTS

#### Инструкция по сборке

0. (Опционально) Установите Android Studio и склонируйте этот репозиторий в новый проект
2. Выполнить шаги из ридми в app/src/main/assets (добавление ресурсов модели)
3. Можно собирать. Если хотите apk то проще всего через CLI: gradlew assembleRelease - тогда apk будет в app/build/outputs/apk/release или через Android Studio, 
тогда можно запустить в виртуалке или на физическом телефоне напрямую.
