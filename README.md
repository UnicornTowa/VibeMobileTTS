### Vosk TTS Mobile

Мобильное приложение на анроид для запуска модели синтеза VoskTTS

#### Инструкция по сборке

0. Установите Android Studio и склонируйте этот репозиторий в новый проект
1. Создать ключ для подписи (это требования андроид, правда). Ключ надо положить в папку keys в корне. В корне проекта создать keystore.properties соответствующий созданному ключу. (Инструкции гуглятся)
2. Выполнить шаги из ридми в app/src/main/assets
3. Можно собирать. Если хотите apk то проще всего через CLI: gradlew assembleRelease
